Detailed Description

This is the detailed description for this class.

This class AnalyzerDriver works as the main driver for a dynamic code analyzer for Java programming language.

This class is responsible for taking a java file path from the user and generating a text file that contains the details of the visited blocks, and generating an html file that contains statement and branches code coverage reports.
